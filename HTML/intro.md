## 支持他的图片格式

可以使用，Photoshop存储为web格式

| 格式   | 说明                 | 备注                           |
| ------ | -------------------- | ------------------------------ |
| Gif    | 支持透明不支持半透明 | 网页中的大图高清图片（体积大） |
| jgp    | 不支持透明，         | 网页中的小图标（动图）         |
| png-8  | 支持透明不支持半透明 | 网页中的小图标                 |
| png-24 | 支持透明支持半透明   | 图像中存在半透明效果           |
