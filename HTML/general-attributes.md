# 通用属性

| 属性   | 作用                                                         | 备注 |
| ------ | ------------------------------------------------------------ | ---- |
| id     | 为元素设置一个唯一的标识                                     |      |
| class  | 为元素设置一组标识 多个名称用空格分割，允许多元素使用同一个class名 |      |
| title  | 显示一段提示信息                                             |      |
| style  | 规定元素样式                                                 |      |
| lang   | 设置元素中内容的语言代码。                                   |      |
| data-* | 用于存储页面的自定义数据                                     |      |

## date用来存储DOM数据无实际作用

date-数据名1-数据名2等来定义



## lang 属性

**语法**

```
<element lang="*language_code*">
```

| 值              | 描述                                                         |
| :-------------- | :----------------------------------------------------------- |
| *language_code* | 规定元素内容的语言代码。[语言代码参考手册](../Appendix/language-code.md)。 |
