# 通用属性

| 属性  | 作用                                                         | 备注 |
| ----- | ------------------------------------------------------------ | ---- |
| id    | 为元素设置一个唯一的标识                                     |      |
| class | 为元素设置一组标识 多个名称用空格分割，允许多元素使用同一个class名 |      |
| title | 显示一段提示信息                                             |      |
