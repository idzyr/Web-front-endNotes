# 超链接

## 统一声明打开方式target属性

```html
在head标签对里加
<base target=”_blank”/> 作用定义页面中所有链接打开的方式

<!-- 如果想要某一个页面从当前页面打开可以加target=“_self” >
```

## 其它功能

### 文件下载

a标签中不再是链接而是文件名就变成下载功能了如；

```html
<a href=”要下载的文件名”>标签名</a>
```

### 锚点

a标签里直接加id可以实现锚点作用如；

```html
<a href=”#(id)”>标签名</a>
```

> **注意；**
>
> ​	a标签锚点作用如果一开始就定义页面内所有链接的打开类型为新建页面打开，需要再锚点后面加target="_self"当前页面打开否则会出现新建页面打开现象

## 伪类样式

```css
a:link{
    color: red;
}
```



1. `link`——为访问（默认）
2. `hover`——鼠标悬停(鼠标划过)
3. `active`——链接激活（鼠标按下）
4. `visited`——访问过后（点击过后）

**a标签设置伪类样式的顺序；**

1. ink{ 样式} （为访问）
2. visited{ 样式}（访问过后）
3. hover{ 样式}（鼠标悬停）
4. active{ 样式}（链接激活）
