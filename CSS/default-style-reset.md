# 默认样式重置

> 目的为了兼容大部分浏览器

- body标签默认有一个8像素的外边框（margin）
- p标签也有一个16像素的上下外边距
- h标题标签也有外边距
- ol和ul都有样式（list-style:none），内外边距也有（padding和margin）
- dl 和 dd都有外边距（margin）
- font-family XX（字体样式）
- inmg（border:none边框）
- th，td{ padding：0px} 表头和单元格
- Form；margin:0
- Input；margin：0 padding：0
- select；margin：0
- textarea；margin：0 padding：0
