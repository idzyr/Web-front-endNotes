# zoom【放大元素】

| 值     | 描述                                                         |
| ------ | ------------------------------------------------------------ |
| auto   | 根据`viewport`来既定当前标签的缩放。                         |
| 数字   | 必须是一个非负数。1表示没有缩放，大于1表示放大的倍数，小于1亦然。 |
| 百分比 | 必须是一个非负的百分比。以100%为基础进行缩放。               |
